# BrowseCart 🛒
BrowseCart is a full-stack web application that helps users discover and explore nearby shops and products easily.  
The platform provides location-based shop browsing, authentication, and a smooth shopping experience with a modern UI.
## 🚀 Features
- 🔐 User Authentication
- 👨‍💼 Admin Dashboard
- 📍 Location-Based Shop Discovery
- 🛍️ Browse Shops and Products
- 🔎 Product Search & Filters
- 🛒 Shopping Experience UI
- ⚡ Responsive Frontend UI
- 🌐 Full Stack MERN Application
- 🔄 REST API Integration
- 📦 MongoDB Database Support
## 🛠️ Tech Stack
### Frontend
- React.js
- CSS
- Bootstrap
### Backend
- Node.js
- Express.js
### Database
- MongoDB
